About

Premium Layersis a clean and professional vcard / resume template, perfect for all types of companies and personal web sites. Built on the Advanced Bootstrap 3 framework with HTML5 / CSS3. Its responsive too - Will fit great on all devices. More features like 5 Homepages, 2 Portfolio pages, About page, Services page, Masakini layouts, Timeline homepages and much more gives you more space to work with and also with a few lines of code.

Clean coded and Easy updation are the key features. Go ahead to know more.

CSS files and structure

I have used Nine CSS files in this theme.

  1. style.css - Contains all of the specific stylings for the page.
  2. bootstrap.css - Bootstrap css.
  3. font-awesome.min.css - Fontawesome Icons.
  4. flexslider.css - Contains all of the specific stylings for Flexslider plugin
  5. nivo-lightbox.css - Contains all of the specific stylings for Nivo lightbox plugin
  6. animate.css - Contains all of the specific stylings for Animation plugin

Javascript

This theme imports Fourteen Javascript files.

  1. jquery.js - To Enable jQuery.
  2. bootstrap.js - Plugin for Bootstrap framework.
  3. html5-shiv.js - Enable HTML5.
  4. respond.min.js - Isotope Plugin.
  5. contact.js - Contact form Plugin.
  6. gmaps.js - Google map Plugin.
  7. jquery.appear.js - Windows scroll Plugin.
  8. jquery.inview.js - Windows scroll Plugin.
  9. jquery.easing.js - Jquery Easing Plugin.
  10. jquery.flexslider.js - Flexslider Plugin.
  11. nivo-lightbox.min.js - Lightbox Plugin.
  12. animations.js - Animations Plugin.
  13. jquery.mixitup.min.js - Mixitup Portfolio Plugin.
  14. main.js - Custom theme js.
